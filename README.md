# Sistema de Cadastramento de Tarifas

Sistema desenvolvido em Java que realiza o cadastramento de uma tarifa de uma operadora de benefícios de vale-transporte. Uma tarifa de uma operadora só pode ser cadastrada caso a operadora não possua uma tarifa de mesmo valor com status ativa dentro de um período de 6 meses.

O sistema possui um repositório com um banco de dados falso onde são persistidas as informações geradas na execução do código. Ao executarmos a classe Main serão criadas algumas operadoras e tarifas para o teste do sistema.
